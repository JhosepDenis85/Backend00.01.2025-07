PREGUNTAS

1. ¿Cómo defines una función?

Respuesta 1: En JavaScript, una función se define utilizando la palabra clave function, seguida del nombre de la función, paréntesis para los parámetros y llaves para el bloque de código. Por ejemplo:

function saludar() {

alert("¡Hola!"); }

También puedes definir funciones como expresiones:

const sumar = function(a, b) {

return a + b; }

O usando la sintaxis de función flecha:

const multiplicar = (a, b) => a * b;


2. ¿Hasta cuantos argumentos puedes declarar en una función?

Respuesta 2: En JavaScript, no hay un límite fijo para la cantidad de argumentos que puedes declarar en una función. Puedes definir tantos parámetros como necesites en la declaración de la función.

Sin embargo, es recomendable mantenerlos en un número razonable para facilitar el mantenimiento del código.
